Enables exclusive appearance to Cross Sell Multiple products on WooCommerce Product Detail page..

== Description ==
<blockquote>Please Note - This plugin requires WooCommerce Version 2.4 or greater in order to properly function.</blockquote>

The WC Ultimate Cross Selling Plugin is ecommerce personalization tool, developed to increase online revenue and sales conversions.

This plugin will allow you to enable “Recommended” Or “Bought Together” products list generated right beside the product description in product detail page. It will ensure customers do not miss to check these ‘Suggested products’.

The WC Ultimate Cross Selling Plugin provides options that you can customize Cross sell product list for any Product. Infect you can even assign Single product or Product variant to fine tune your “Recommended products list” which is very distinct and unique feature of this plugin.

== Installation ==
1. Upload the folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= This plugin doesn't do anything =
WC Ultimate Cross Selling is an extension of WooCommerce. Without at least WooCommerce version 2.4 installed and active then there will be no need for this plugin

= After activation I now get a Fatal Error?
This is due to an outdated version of WooCommerce. If you update to the most recent version of WooCommerce your issue will be fixed.
